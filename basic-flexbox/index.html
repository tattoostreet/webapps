Basic Flexbox

Random Tips

  • div.item*3{Items $} - Create multiple HTML elements with defined class
  • diplay: float; - legacy method before flexbox
  • display: flex; justify-content: center; align-items: center; - center a div by adding flex to items (nested flex)
  • .container :nth-child()
  • justify-content vs align-items - justify is always relative to main axis (vertical) while align-items is with cross axis (horizontal)
  • flex-start vs start - flex-start takes into account the presence of the -reverse values of the flex direction, while start does not
